## 1. 🔆 บทนำ — ทำไมการคำนวณขนาดถึงสำคัญ

หลายคนติดโซลาร์แล้วเจอปัญหา — **"ทำไมผลิตไฟได้น้อยกว่าที่คุยไว้?"** หรือ **"ทำไมค่าไฟยังสูงอยู่?"** หรือแม้แต่ **"ทำไมการไฟฟ้าปฏิเสธไม่ให้ติดตั้ง?"**

คำตอบส่วนใหญ่อยู่ที่การ **คำนวณขนาดระบบผิด**

การคำนวณขนาดระบบโซลาร์ไม่ใช่แค่การหารเลขง่ายๆ ต้องเข้าใจ:

- **พฤติกรรมการใช้ไฟ** ของบ้านคุณ — ใช้เท่าไหร่ ช่วงไหน เท่าไหร่
- **ศักยภาพของหลังคา** — ได้แสงเท่าไหร่ ทิศไหน มีเงาบังไหม
- **ข้อจำกัดทางเทคนิค** — Inverter รับได้เท่าไหร่ การไฟฟ้าจำกัดเท่าไหร่
- **เป้าหมายทางการเงิน** — อยากประหยัดเท่าไหร่ คืนทุนกี่ปี

ถ้าคุณคำนวณผิด ผลที่ตามมาคือ:

**กรณีติดขนาดเล็กเกินไป:**
- ผลิตไฟได้น้อย ค่าไฟลดน้อยมาก คุ้มค่าไม่คุ้ม
- คืนทุนช้า หรือไม่คุ้มเลย
- เสียดายพื้นที่หลังคาที่ติดไป

**กรณีติดขนาดใหญ่เกินไป:**
- การไฟฟ้าอาจปฏิเสธไม่ให้ติดตั้ง
- ขายคืนคุ้มค่าน้อย (ราคารับซื้อต่ำกว่าซื้อ)
- ลงทุนเกินจำเป็น ใช้เวลาคืนทุนนานขึ้น

บทความนี้จะพาคุณ **คำนวณด้วยตัวเองอย่างละเอียด** ไม่ต้องง้อใคร พร้อมตัวอย่างจริง 5 แบบ และเครื่องมือที่ช่วยให้คำนวณได้แม่นยำขึ้น

## 2. 📊 ข้อมูลที่ต้องรู้ก่อนคำนวณ

ก่อนจะคำนวณขนาดระบบ คุณต้องรู้ข้อมูลเหล่านี้ก่อน:

### 2.1 ข้อมูลที่ 1: ค่าไฟเฉลี่ยต่อเดือน

ดูจากบิลค่าไฟ 3-6 เดือนล่าสุด แล้วหาค่าเฉลี่ย

**วิธีหา:**
```
ค่าไฟรวม 3-6 เดือน ÷ จำนวนเดือน = ค่าไฟเฉลี่ย/เดือน
```

**ตัวอย่าง:** ค่าไฟ 3 เดือน = 3,200 + 3,800 + 4,100 = 11,100 บาท
```
11,100 ÷ 3 = 3,700 บาท/เดือน (ค่าเฉลี่ย)
```

**ทำไมต้องดูหลายเดือน:**
- ค่าไฟแต่ละเดือนไม่เท่ากัน เพราะฤดูกาล คนอยู่บ้านไม่เท่ากัน
- ถ้าดูแค่เดือนเดียวอาจผิดพลาดมาก
- ควรดูทั้งฤดูร้อนและฤดูฝน

### 2.2 ข้อมูลที่ 2: จำนวนหน่วยที่ใช้ต่อเดือน (kWh)

ดูจากบิลไฟ จะมีบอก "ปริมาณการใช้ kWh" ต่างจากค่าไฟบาท

**ความสัมพันธ์ระหว่างค่าไฟบาท กับ หน่วย kWh:**
```
ค่าไฟต่อหน่วย = ค่าไฟรวม ÷ จำนวนหน่วย
```

สำหรับประเทศไทย (อัตราบ้านอยู่อาศัย ปกติ):
- ค่าไฟต่อหน่วย (ไม่รวม Ft และ VAT): ประมาณ 3.50-4.50 บาท/kWh
- ค่า Ft (ค่าไฟฟ้าผันแปร): ประมาณ 0.50-1.00 บาท/kWh
- VAT 7%: เพิ่มอีกประมาณ 7% ของค่าไฟรวม

**ตัวอย่าง:** ใช้ไฟ 350 kWh/เดือน ค่าไฟ 3,700 บาท
```
ค่าต่อหน่วย = 3,700 ÷ 350 = 10.57 บาท/kWh (รวม Ft และ VAT แล้ว)
```

### 2.3 ข้อมูลที่ 3: พื้นที่หลังคาที่ใช้ได้

วัดพื้นที่หลังคาที่โดนแดดเต็มๆ อย่างน้อย 4-6 ชั่วโมง/วัน

**วิธีวัด:**
1. ถ่ายรูปหลังคาตอนเที่ยงวัน (12:00-13:00) เพื่อดูว่าแดดเต็มที่ไหม
2. วัดขนาดหลังคาที่จะติดโซลาร์ (ความยาว × ความกว้าง)
3. หักพื้นที่ที่มีเงาบัง เช่น ปล่องไฟ สกายไลท์ ต้นไม้

**สูตรคร่าวๆ พื้นที่ที่ต้องการ:**
```
พื้นที่ (ตร.ม.) = จำนวนแผง × 2.0 ตร.ม./แผง (สำหรับแผงขนาด 1.7 × 1.0 ม.)
```

**ตัวอย่าง:** ต้องการติด 10 แผง
```
พื้นที่ = 10 × 2.0 = 20 ตร.ม.
```

### 2.4 ข้อมูลที่ 4: ทิศทางหลังคา

ทิศทางหลังคามีผลต่อปริมาณแสงแดดที่ได้รับ:

| ทิศทาง | แสงแดดที่ได้ | คะแนน |
|--------|-------------|--------|
| **ใต้** | มากที่สุดในประเทศไทย | ⭐⭐⭐⭐⭐ |
| **ตะวันออกเฉียงใต้** | มาก | ⭐⭐⭐⭐ |
| **ตะวันตกเฉียงใต้** | มาก | ⭐⭐⭐⭐ |
| **ตะวันออก** | ปานกลาง (เช้า) | ⭐⭐⭐ |
| **ตะวันตก** | ปานกลาง (บ่าย) | ⭐⭐⭐ |
| **เหนือ** | น้อยที่สุด | ⭐ |

**หมายเหตุ:** ประเทศไทยอยู่ในเขตซีกโลกเหนือ ดังนั้นหลังคาที่หันไปทางใต้จะได้แสงแดดมากที่สุดตลอดปี

### 2.5 ข้อมูลที่ 5: ชั่วโมงแสงแดดเฉลี่ยต่อวัน (Sun Hours)

ชั่วโมงแสงแดดเฉลี่ยต่อวัน (Peak Sun Hours) คือจำนวนชั่วโมงที่แสงแดดมีความเข้มเท่ากับ 1,000 W/m²

**สำหรับประเทศไทย:**

| ภูมิภาค | ชั่วโมงแดดเฉลี่ย/วัน |
|---------|---------------------|
| กรุงเทพฯ และภาคกลาง | 4.5 - 5.5 ชม. |
| ภาคตะวันออก | 5.0 - 5.5 ชม. |
| ภาคใต้ (ฝั่งอันดามัน) | 4.0 - 5.0 ชม. |
| ภาคใต้ (ฝั่งอ่าวไทย) | 4.5 - 5.0 ชม. |
| ภาคเหนือ | 4.5 - 5.0 ชม. |
| ภาคตะวันออกเฉียงเหนือ | 4.5 - 5.5 ชม. |

**หมายเหตุ:** ค่านี้เป็นค่าเฉลี่ยตลอดทั้งปี ฤดูร้อนจะสูงกว่า (5-6 ชม.) ฤดูฝนจะต่ำกว่า (3-4 ชม.)

### 2.6 ข้อมูลที่ 6: เงาบัง (Shading)

เงาบังมีผลมากต่อการผลิตไฟของโซลาร์ ถ้ามีเงาบังแม้เพียงเล็กน้อย ก็จะลดประสิทธิภาพได้มาก

**วิธีตรวจสอบเงาบัง:**

1. **ถ่ายรูปหลังคาทุก 2 ชั่วโมง ตั้งแต่ 8 โมงถึง 4 โมง** — ดูว่ามีเงาบังตอนไหนบ้าง
2. **ใช้แอป Solar Pathfinder หรือ Sun Seeker** — วัดมุมและทิศทางที่แดดบล็อก
3. **สังเกตต้นไม้หรืออาคารรอบบ้าน** — ดูว่ามีอะไรบังหรือเปล่า

**ถ้ามีเงาบัง:**
- น้อยกว่า 5% ของวัน → ไม่ต้องกังวลมาก
- 5-10% ของวัน → ลดขนาดระบบลง 5-10%
- มากกว่า 10% ของวัน → ควรพิจารณาติดตำแหน่งอื่น หรือติด Micro-Inverter แทน String Inverter

### 2.7 ข้อมูลที่ 7: โครงสร้างหลังคา

หลังคาต้องแข็งแรงพอที่จะรับน้ำหนักของโซลาร์และขายึด

**น้ำหนักของโซลาร์ต่อ 1 ตร.ม.:**
- แผงโซลาร์ + ขายึด: ประมาณ 15-25 kg/ตร.ม.

**ตรวจสอบหลังคา:**
- หลังคาคอนกรีต: รับน้ำหนักได้ดี ไม่ต้องเสริมโครงสร้าง
- หลังคาเพล็กซ์ (Metal Sheet): ต้องตรวจสอบว่าแข็งแรงพอ อาจต้องเสริมโครงสร้าง
- หลังคาไม้: ต้องประเมินโดยวิศวกรก่อน
- หลังคาแบน (Flat Roof): ติดตั้งได้ แต่ต้องใช้ระบบ Ballast หรือเจาะยึด

## 3. 🔢 สูตรคำนวณพื้นฐาน

### 3.1 สูตรที่ 1: คำนวณขนาดระบบที่ต้องการ (kWp)

**สูตร:**
```
ขนาดระบบ (kWp) = จำนวน kWh ที่ใช้/เดือน
                   ÷ จำนวนวันในเดือน (30)
                   ÷ ชั่วโมงแดดเฉลี่ย/วัน
                   × ค่าปรับ (Loss Factor)
```

**ค่าปรับ (Loss Factor):**
- ประมาณ 1.2-1.3 เพื่อชดเชยความสูญเสียต่างๆ
- รวมความสูญเสียจาก: Inverter (3-5%), สายไฟ (1-2%), ฝุ่น/เงา (5-10%), อุณหภูมิ (5-10%)

**ตัวอย่าง:**
```
ใช้ไฟ 300 kWh/เดือน
ชั่วโมงแดดเฉลี่ย = 4.5 ชม./วัน
ขนาดระบบ = 300 ÷ 30 ÷ 4.5 × 1.25
          = 10 ÷ 4.5 × 1.25
          = 2.22 × 1.25
          = 2.78 kWp
→ ใช้ระบบ 3 kWp
```

### 3.2 สูตรที่ 2: คำนวณว่าต้องใช้แผงกี่แผง?

**สูตร:**
```
จำนวนแผง = ขนาดระบบ (W) ÷ ขนาดแผง (W)
```

**ตัวอย่าง:**
```
ต้องการ 3,000 W (3 kWp)
ใช้แผง 600W = 3000 ÷ 600 = 5 แผง
ใช้แผง 550W = 3000 ÷ 550 = 5.45 → ใช้ 6 แผง
ใช้แผง 450W = 3000 ÷ 450 = 6.67 → ใช้ 7 แผง
```

**ข้อควรระวัง:** อย่าปัดเศษลง! ถ้าได้ 5.45 แผง ต้องใช้ 6 แผง เพราะแผงครึ่งไม่มีในโลกความจริง

### 3.3 สูตรที่ 3: คำนวณพื้นที่ที่ต้องใช้

**สูตร:**
```
พื้นที่ (ตร.ม.) = จำนวนแผง × 2.0 ตร.ม./แผง (เผื่อพื้นที่ + ช่องหายใจ)
```

**ตัวอย่าง:**
```
5 แผง × 2.0 = 10 ตร.ม.
6 แผง × 2.0 = 12 ตร.ม.
7 แผง × 2.0 = 14 ตร.ม.
```

### 3.4 สูตรที่ 4: คำนวณว่าผลิตไฟได้เท่าไหร่ต่อเดือน

**สูตร:**
```
ผลิตไฟ/เดือน (kWh) = ขนาดระบบ (kWp)
                      × ชั่วโมงแดดเฉลี่ย/วัน
                      × จำนวนวันในเดือน (30)
                      × ประสิทธิภาพรวม (0.75-0.85)
```

**ตัวอย่าง:**
```
ระบบ 3 kWp, ชั่วโมงแดด 4.5 ชม., ประสิทธิภาพ 0.80
ผลิตไฟ/เดือน = 3 × 4.5 × 30 × 0.80
              = 324 kWh/เดือน
```

### 3.5 สูตรที่ 5: คำนวณว่าลดค่าไฟได้เท่าไหร่

**สูตร:**
```
ประหยัดค่าไฟ/เดือน (บาท) = ผลิตไฟได้ (kWh) × ค่าไฟต่อหน่วย (บาท)
```

**ตัวอย่าง:**
```
ผลิตไฟได้ 324 kWh/เดือน
ค่าไฟ 4.18 บาท/kWh
ประหยัด = 324 × 4.18 = 1,354 บาท/เดือน
```

### 3.6 สูตรที่ 6: คำนวณระยะเวลาคืนทุน

**สูตร:**
```
คืนทุน (ปี) = ค่าใช้จ่ายรวม (บาท) ÷ ประหยัดค่าไฟ/ปี (บาท)
```

**ตัวอย่าง:**
```
ค่าใช้จ่ายรวม = 120,000 บาท
ประหยัดค่าไฟ/เดือน = 1,354 บาท
ประหยัด/ปี = 1,354 × 12 = 16,248 บาท
คืนทุน = 120,000 ÷ 16,248 = 7.4 ปี
```

## 4. 📝 ตัวอย่างการคำนวณจริง 5 แบบ

### 4.1 กรณีศึกษาที่ 1: คอนโดเล็กในกรุงเทพฯ

**โปรไฟล์:**
- ค่าไฟเฉลี่ย: 1,500 บาท/เดือน
- ใช้ไฟ: 150 kWh/เดือน
- อยู่: คอนโด 1 ห้องนอน ชั้น 15
- หลังคา: ทาวน์เฮาส์ หลังคาแบน 8 ตร.ม. (เฉพาะส่วนที่ใช้ได้ 5 ตร.ม.)
- ทิศ: ตะวันออก
- เงาบัง: มีตึกสูงบังบ้าง 10% ของวัน

**วิเคราะห์:**
- พื้นที่จำกัด: ติดได้สูงสุด 2-3 แผง (4-6 ตร.ม.)
- ทิศไม่ดีที่สุด: ตะวันออก ได้แดดเช้า แต่ไม่ถึงบ่าย
- ชั่วโมงแดดเฉลี่ย: ประมาณ 4.0 ชม./วัน (หลังหักเงาบัง)

**คำนวณ:**
```
ขนาดระบบ = 150 ÷ 30 ÷ 4.0 × 1.25 = 1.56 kWp → ใช้ 1.5 kWp
แผง 550W = 1500 ÷ 550 = 2.73 → ใช้ 3 แผง (1.65 kWp)
พื้นที่ = 3 × 2.0 = 6 ตร.ม. (พอดีกับที่มี)
```

**ผลลัพธ์:**
- ระบบ **1.65 kWp** (3 แผง × 550W)
- ผลิตไฟได้: 1.65 × 4.0 × 30 × 0.80 = **158 kWh/เดือน**
- ประหยัด: 158 × 4.18 = **660 บาท/เดือน**
- ลดค่าไฟ: 44% จากเดิม
- ค่าใช้จ่ายประมาณ: 50,000-60,000 บาท
- คืนทุน: 7-9 ปี

**ข้อสรุป:** ไม่คุ้มมากสำหรับคอนโด เพราะพื้นที่จำกัด ทิศไม่ดี คุ้มกว่าการประหยัดไฟด้วยวิธีอื่น

---

### 4.2 กรณีศึกษาที่ 2: บ้านเดี่ยว 2 ห้องนอน ในเมือง

**โปรไฟล์:**
- ค่าไฟเฉลี่ย: 3,500 บาท/เดือน
- ใช้ไฟ: 350 kWh/เดือน
- อยู่: บ้านเดี่ยว 2 ชั้น หลังคา 60 ตร.ม. (ใช้ได้ 40 ตร.ม.)
- ทิศ: ใต้-ตะวันออก
- เงาบัง: น้อย (ต้นไม้เล็กน้อย) ประมาณ 5%
- ชั่วโมงแดดเฉลี่ย: 4.5 ชม./วัน

**วิเคราะห์:**
- พื้นที่เพียงพอ: ติดได้สูงสุด 15-20 แผง (30-40 ตร.ม.)
- ทิศดี: ใต้ ดีที่สุด
- เงาบังน้อย: ประสิทธิภาพสูง

**คำนวณ:**
```
ขนาดระบบ = 350 ÷ 30 ÷ 4.5 × 1.25 = 3.24 kWp → ใช้ 3 kWp
แผง 550W = 3000 ÷ 550 = 5.45 → ใช้ 6 แผง (3.3 kWp)
พื้นที่ = 6 × 2.0 = 12 ตร.ม.
ตรวจสอบ: พื้นที่ที่มี 40 ตร.ม. > 12 ตร.ม. ✓
```

**ผลลัพธ์:**
- ระบบ **3.3 kWp** (6 แผง × 550W)
- ผลิตไฟได้: 3.3 × 4.5 × 30 × 0.80 = **356 kWh/เดือน**
- ประหยัด: 356 × 4.18 = **1,488 บาท/เดือน**
- ลดค่าไฟ: 42% จากเดิม
- ค่าใช้จ่ายประมาณ: 90,000-110,000 บาท
- คืนทุน: 6-7 ปี

**ข้อสรุป:** คุ้มค่ามาก! ระบบขนาดเหมาะสม พื้นที่เพียงพอ คืนทุนเร็ว

### 4.5 กรณีศึกษาที่ 5: บ้านเก่าในชนบท ตะวันออกเฉียงเหนือ

- ค่าไฟเฉลี่ย: 2,000 บาท/เดือน
- ใช้ไฟ: 200 kWh/เดือน
- อยู่: บ้านเดี่ยวเก่า หลังคา 45 ตร.ม. (ใช้ได้ 25 ตร.ม.)
- ทิศ: เหนือ (ทิศที่แย่ที่สุด)
- เงาบัง: มีต้นไม้ใหญ่บัง 15% ของวัน
- ชั่วโมงแดดเฉลี่ย: 3.5 ชม./วัน (หลังหักทิศเหนือ + เงาบัง)
- ไม่มีไฟฟ้าถึง: ต้องการ Off-Grid

**วิเคราะห์:**
- ทิศเหนือ: ไม่แนะนำติดโซลาร์ แต่ถ้าเป็นทางเลือกเดียว ก็ต้องยอมรับว่าผลิตได้น้อย
- พื้นที่พอใช้: ติดได้ประมาณ 10-12 แผง
- ต้องการ Off-Grid: เพราะไม่มีไฟฟ้าถึง

**คำนวณ (Off-Grid):**
```
ขนาดระบบ = 200 ÷ 30 ÷ 3.5 × 1.30 = 2.47 kWp → ใช้ 2.5 kWp
แผง 550W = 2500 ÷ 550 = 4.55 → ใช้ 5 แผง (2.75 kWp)
พื้นที่ = 5 × 2.0 = 10 ตร.ม.
ตรวจสอบ: พื้นที่ที่มี 25 ตร.ม. > 10 ตร.ม. ✓

แบตเตอรี่ (Off-Grid):
ต้องการเก็บ 2 วัน: 200 × 2 = 400 Wh × ประสิทธิภาพ 0.85 = 470 kWh
ใช้ Lithium 48V: ความจุ = 470 ÷ 48 = 9.8 Ah → ใช้ 100 Ah
แบตเตอรี่ 100Ah 48V LiFePO4: ราคา 40,000-60,000 บาท
```

**ผลลัพธ์ (Off-Grid):**
- ระบบ **2.75 kWp** (5 แผง × 550W)
- แบตเตอรี่ 100Ah 48V
- Generator สำรอง (Diesel 3kVA): ราคา 20,000-30,000 บาท
- ผลิตไฟได้: 2.75 × 3.5 × 30 × 0.75 = **217 kWh/เดือน** (น้อยกว่าที่ใช้เล็กน้อย)
- ประหยัด: 217 × 4.18 = **907 บาท/เดือน** (แต่ไม่ต้องจ่ายค่าไฟการไฟฟ้าเลย)
- ค่าใช้จ่ายรวม: 120,000-160,000 บาท
- คืนทุน: เทียบกับต่อไฟฟ้าใหม่ (ค่าต่อมิเตอร์ + สาย) ประมาณ 100,000-150,000 บาท บวกค่าไฟตลอดชีวิต คุ้มค่ากว่ามาก

**ข้อสรุป:** คุ้มค่ากว่าการไม่ติดโซลาร์เลย แม้ทิศไม่ดี แต่ถ้าเป็นทางเลือกเดียวที่จะมีไฟใช้ ก็คุ้มค่าที่จะลงทุน

### 5.3 สิ่งที่ราคาควรรวม

✅ **รวมในราคาติดตั้ง:**
- แผงโซลาร์ (จำนวนตามขนาด)
- Inverter (String หรือ Hybrid)
- ขายึดแผง (Mounting)
- สายไฟ DC, AC

- การติดตั้งและทดสอบ

❌ **ไม่รวมในราคาติดตั้ง:**
- การขออนุญาตการไฟฟ้า (500-2,000 บาท)
- Bidirectional Meter (ถ้าต้องการขายไฟคืน) (3,000-8,000 บาท)
- การเสริมโครงสร้างหลังคา (ถ้าจำเป็น)
- Battery (ถ้าเป็น Hybrid หรือ Off-Grid)
- Generator (สำหรับ Off-Grid)

### 5.4 วิธีเช็คว่าราคาที่ได้รับเป็นราคาที่เหมาะสม

**เทียบราคา 2-3 บริษัท:** อย่าตกลงกับบริษัทแรกที่เสนอราคา

**เช็คว่าราคาต่อ kWp เป็นอย่างไร:**
```
ราคาต่อ kWp = ราคารวม ÷ ขนาดระบบ (kWp)
```

ตัวอย่าง: เสนอราคา 150,000 บาท สำหรับระบบ 5 kWp
```
ราคาต่อ kWp = 150,000 ÷ 5 = 30,000 บาท/kWp
```
ถ้าเกิน 35,000-40,000 บาท/kWp = แพงไป
ถ้าต่ำกว่า 22,000-25,000 บาท/kWp = น่าสงสัย อาจใช้อุปกรณ์คุณภาพต่ำ

## 7. ⚠️ ข้อผิดพลาดที่พบบ่อย

### 7.1 ข้อผิดพลาดที่ 1: คำนวณจากค่าไฟเดือนเดียว

**ผิด:** ดูค่าไฟแค่เดือนที่แพงที่สุด แล้วคำนวณว่าต้องติดเท่าไหร่ถึงจะลดค่าไฟเหลือ 0

**จริง:** ค่าไฟแต่ละเดือนไม่เท่ากัน เช่น เดือนที่ร้อนมาก ค่าไฟจะสูงกว่าเดือนที่ฝนตก

**วิธีถูก:** ใช้ค่าเฉลี่ย 3-6 เดือน เพื่อให้คำนวณได้ใกล้เคียงความจริง

---

### 7.2 ข้อผิดพลาดที่ 2: ติดขนาดใหญ่เกินเพื่อให้ค่าไฟเหลือ 0

**ผิด:** อยากให้ค่าไฟเหลือ 0 บาท ก็เลยติดระบบใหญ่มาก

**จริง:** 
- การไฟฟ้าจำกัดขนาดระบบที่ติดได้ (ปกติไม่เกิน 110% ของการใช้)
- ขายคืนคุ้มค่าน้อย (ราคารับซื้อต่ำกว่าซื้อ)
- ลงทุนเกินจำเป็น ใช้เวลาคืนทุนนานขึ้น

**วิธีถูก:** ติดขนาดที่เหมาะสม (70-100% ของการใช้) แล้วใช้ไฟให้เต็มที่ก่อน

---

### 7.3 ข้อผิดพลาดที่ 3: ไม่เช็คพื้นที่หลังคา

**ผิด:** คำนวณว่าต้องติด 10 แผง แต่พอไปดูหลังคาจริง ติดได้แค่ 6 แผง

**จริง:** พื้นที่ที่วัดได้จากแปลน ≠ พื้นที่ที่ใช้ได้จริง ต้องเช็ค:
- มีเงาบังตรงไหน
- มีปล่องไฟ สกายไลท์ หรือเสาอากาศบังไหม
- เว้นระยะจากขอบหลังคาอย่างน้อย 50 ซม.

**วิธีถูก:** ไปวัดพื้นที่หลังคาจริง ถ่ายรูปตอนเที่ยงวัน แล้วค่อยคำนวณ

---

### 7.4 ข้อผิดพลาดที่ 4: ใช้ชั่วโมงแดดเฉลี่ยสูงเกินไป

**ผิด:** ใช้ค่า 6-7 ชม./วัน เพราะอ่านเจอว่าประเทศไทยแดดแรง

**จริง:** 
- ชั่วโมงแดดเฉลี่ยจริง 4-5 ชม./วัน (เป็นค่าเฉลี่ยทั้งปี)
- ฤดูฝน ชั่วโมงแดดอาจเหลือ 2-3 ชม./วัน
- อุณหภูมิสูงทำให้ประสิทธิภาพลดลง 5-10%

**วิธีถูก:** ใช้ค่า 4-4.5 ชม./วัน แล้วคูณด้วย Loss Factor 1.2-1.3

---

### 7.5 ข้อผิดพลาดที่ 5: เชื่อราคาที่ถูกเกินไป

**ผิด:** ได้รับ предложение ระบบ 5 kWp ในราคา 70,000 บาท ดีใจว่าถูกมาก

**จริง:**
- ราคาต่ำผิดปกติ อาจเป็นอุปกรณ์คุณภาพต่ำ
- แผงโซลาร์กำลังการผลิตต่ำ หรือ Inverter รีแบรนด์
- ไม่มีการรับประกันที่ดี

**วิธีถูก:** เช็คว่า:
- แผงโซลาร์ยี่ห้ออะไร? มีมาตรฐานอะไร? (IEC, TISI)
- Inverter ยี่ห้ออะไร? รับประกันกี่ปี?
- บริษัททำมานานกี่ปี? มีศูนย์บริการที่ไหน?

**สเปกขั้นต่ำที่ควรมี:**
- แผงโซลาร์: กำลังการผลิต 400W+ ต่อแผง มีมาตรฐาน IEC 61215
- Inverter: ยี่ห้อดัง (SMA, Fronius, Growatt, Huawei) รับประกัน 5-10 ปี
- ขายึด: อลูมิเนียม หรือ สแตนเลส ทนสนิม

---

### 7.6 ข้อผิดพลาดที่ 6: ไม่คำนึงถึงการขยายในอนาคต

**ผิด:** ติดระบบพอดีกับการใช้ปัจจุบัน ไม่เผื่อที่ว่างไว้

**จริง:** อาจต้องการใช้ไฟเพิ่มในอนาคต (ซื้อรถไฟฟ้า, ติดแอร์เพิ่ม, ขยายบ้าน)

**วิธีถูก:** 
- ถ้าพื้นที่หลังคาพอ: เผื่อที่ว่างไว้ 20-30% สำหรับติดเพิ่ม
- เลือก Inverter ที่รองรับการขยาย (บางรุ่นต้องเปลี่ยนถ้าเพิ่มแผง)
- ถ้าติด String Inverter: ต้องดูว่า Input ของ Inverter รองรับกี่ String

## 8. 📌 สรุป

**การคำนวณขนาดระบบโซลาร์ไม่ยาก ถ้าเข้าใจหลักการ:**

**ขั้นตอนสรุป:**
1. **รู้ค่าไฟเฉลี่ย/เดือน** → ดูจากบิล 3-6 เดือน
2. **รู้จำนวน kWh ที่ใช้/เดือน** → ดูจากบิล
3. **รู้พื้นที่หลังคาที่ใช้ได้** → วัดจริง ดูเงาบัง
4. **รู้ทิศทางและชั่วโมงแดด** → ใช้ค่าเฉลี่ย 4-5 ชม./วัน
5. **คำนวณขนาดระบบ** → ใช้สูตรข้างบน
6. **เช็คว่าพื้นที่เพียงพอ** → พื้นที่ ≥ จำนวนแผง × 2 ตร.ม.
7. **เช็คว่าคุ้มค่า** → เทียบราคา 2-3 บริษัท

**สรุปง่ายๆ ตามขนาด:**

| ค่าไฟ/เดือน | ขนาดระบบแนะนำ | จำนวนแผง (550W) | ค่าใช้จ่ายโดยประมาณ | ประหยัด/เดือน |
|-------------|--------------|---------------|-------------------|--------------|
| 1,000-2,000 บ. | 1-2 kWp | 2-4 แผง | 40,000-70,000 บ. | 400-800 บ. |
| 2,000-4,000 บ. | 3-5 kWp | 5-9 แผง | 80,000-150,000 บ. | 800-1,600 บ. |
| 4,000-6,000 บ. | 5-8 kWp | 9-14 แผง | 130,000-220,000 บ. | 1,600-2,400 บ. |
| 6,000-10,000 บ. | 8-12 kWp | 14-22 แผง | 200,000-350,000 บ. | 2,400-4,000 บ. |
| 10,000+ บ. | 15+ kWp | 27+ แผง | 350,000+ บ. | 4,000+ บ. |

**คำแนะนำสุดท้าย:**
- **อย่ารีบตัดสินใจ** — เช็คข้อมูลหลายที่ ขอใบเสนอราคา 2-3 บริษัท
- **อย่าหลงราคาถูก** — คุณภาพอุปกรณ์สำคัญกว่าราคา
- **คำนวณให้รอบคอบ** — ถ้าคำนวณผิด จะเสียเงินโดยไม่จำเป็น
- **ปรึกษาผู้เชี่ยวชาญ** — ถ้าไม่แน่ใจ ขอความเห็นจากวิศวกรไฟฟ้า
