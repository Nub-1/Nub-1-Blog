## 1. 🔆 บทนำ

หลายคนติดโซลาร์แล้วเจอปัญหา — "ทำไมผลิตไฟได้น้อยกว่าที่คุยไว้?" หรือ "ทำไมค่าไฟยังสูงอยู่?"

คำตอบส่วนใหญ่อยู่ที่การ **คำนวณขนาดระบบผิด**

บทความนี้จะพาคุณคำนวณด้วยตัวเอง ไม่ต้องง้อใคร

---

## 2. 📊 ข้อมูลที่ต้องรู้ก่อนคำนวณ

### 📌 ข้อมูลที่ 1: ค่าไฟเฉลี่ยต่อเดือน
ดูจากบิลค่าไฟ 3-6 เดือนล่าสุด แล้วหาค่าเฉลี่ย

**วิธีหา:**
```
ค่าไฟรวม 3 เดือน ÷ 3 = ค่าไฟเฉลี่ย/เดือน
```

### 📌 ข้อมูลที่ 2: จำนวนหน่วยที่ใช้ต่อเดือน (Unit)
ดูจากบิลไฟ จะมีบอก "ปริมาณการใช้ kWh"
- ค่าไฟต่อหน่วย = ประมาณ 4.18 บาท (ยังไม่รวม Ft)

### 📌 ข้อมูลที่ 3: พื้นที่หลังคาที่ใช้ได้
- วัดพื้นที่หลังคาที่โดนแดดเต็มๆ
- ความยาว × ความกว้าง = พื้นที่ (ตร.ม.)
- ถ่ายรูปหลังคาตอนเที่ยงวัน ดูว่าแดดเต็มที่ไหม

### 📌 ข้อมูลที่ 4: ทิศทางหลังคา
- ใต้-ใต้: ดีที่สุด
- ตะวันออก/ตะวันตก: พอใช้
- เหนือ: ไม่แนะนำ

## 3. 🔢 สูตรคำนวณพื้นฐาน

### สูตรที่ 1: คำนวณขนาดระบบที่ต้องการ (kWp)

```
ขนาดระบบ (kWp) = จำนวน kWh ที่ใช้/เดือน
                   ÷ 30 วัน
                   ÷ ชั่วโมงแดดเฉลี่ย/วัน (ประมาณ 4-5 ชม.)
                   × 1.2 (ความสูญเสียประมาณ 20%)
```

**ตัวอย่าง:**
```
ใช้ไฟ 300 kWh/เดือน
= 300 ÷ 30 ÷ 4.5 × 1.2
= 2.22 × 1.2
= 2.67 kWp
→ ใช้ระบบ 3 kWp
```

### สูตรที่ 2: คำนวณว่าต้องใช้แผงกี่แผง?

```
จำนวนแผง = ขนาดระบบ (W) ÷ ขนาดแผง (W)
```

**ตัวอย่าง:**
```
ต้องการ 3,000 W (3 kWp)
ใช้แผง 600W = 3000 ÷ 600 = 5 แผง
```

### สูตรที่ 3: คำนวณพื้นที่ที่ต้องใช้

```
พื้นที่ = จำนวนแผง × 2.0 ตร.ม./แผง
```

**ตัวอย่าง:**
```
5 แผง × 2.0 = 10 ตร.ม.
```

## 4. 📝 ตัวอย่างการคำนวณจริง

### กรณีศึกษา: บ้านเดี่ยว 2 ห้องนอน

**ข้อมูล:**
- ค่าไฟเฉลี่ย: 3,500 บาท/เดือน
- ใช้ไฟ: 300 kWh/เดือน
- หลังคา: ใต้ 20 ตร.ม.
- วันละ 4.5 ชั่วโมงแดด

**คำนวณ:**
```
ขนาดระบบ = 300 ÷ 30 ÷ 4.5 × 1.2 = 2.67 kWp → ใช้ 3 kWp
จำนวนแผง = 3,000 ÷ 600 = 5 แผง
พื้นที่ = 5 × 2.0 = 10 ตร.ม. (จาก 20 ตร.ม.ที่มี → ใช้ได้)
```

**ผลลัพธ์:**
- ระบบ 3 kWp (5 แผง × 600W)
- คาดว่าผลิตได้: 300-350 kWh/เดือน
- ลดค่าไฟได้: 1,200-1,400 บาท/เดือน
- คืนทุน: 5-7 ปี

## 5. 💡 ประมาณการค่าใช้จ่าย

### ราคาต่อ 1 kWp (พร้อมติดตั้ง)

| ระดับ | ราคา/kWp | ระบบ 3kWp |
|-------|---------|-----------|
| ระบบดีมาก | 30,000-35,000 | 90,000-105,000 |
| ระบบดี | 25,000-30,000 | 75,000-90,000 |
| ระบบประหยัด | 20,000-25,000 | 60,000-75,000 |

### สิ่งที่ราคาควรรวม
- แผงโซลาร์
- Inverter
- การติดตั้ง
- สายไฟ, Fuse, DC Breaker
- การขออนุญาต (ถ้ามี)

### สิ่งที่มักไม่รวม
- Battery (ถ้าเป็น On-Grid)
- การซ่อมหลังคา/โครงสร้าง
- ปั๊มน้ำ, อุปกรณ์เพิ่มเติม

## 6. ⚠️ ข้อควรระวัง

### 🚩 อย่าติดเกินขนาด
- ถ้าติดเกิน 110% ของการใช้ การไฟฟ้าอาจปฏิเสธ
- ขายคืนราคาต่ำกว่าซื้อ คุ้มค่าไม่เท่ากัน

### 🚩 ดูเรื่อง shading
- มีต้นไม้/อาคารบังบ้างไหม?
- ถ้าบังเกิน 10% ของวัน → ลดขนาดลง

### 🚩 เช็คน้ำหนักหลังคา
- แผงโซลาร์ + ขาติดตั้ง หนักประมาณ 15-20 kg/ตร.ม.
- หลังคาคอนกรีตรับน้ำหนักได้
- หลังคาเพล็กซ์ ต้องเสริมโครงสร้าง

## 7. 📌 สรุป

**การคำนวณขนาดระบบไม่ยาก:**

1. **รู้ค่าไฟเฉลี่ย/เดือน** → ดูจากบิล
2. **คำนวณ kWp ที่ต้องการ** → ใช้สูตรข้างบน
3. **เช็คพื้นที่หลังคา** → ดูว่าพอไหม
4. **เช็คทิศทาง** → ใต้หรือตะวันออก-ตะวันตกดี
5. **ขอใบเสนอราคา** → เทียบ 2-3 บริษัท

**สรุปง่ายๆ:**
- ค่าไฟ 2,000 บ → ระบบ 2-3 kWp
- ค่าไฟ 4,000 บ → ระบบ 5-6 kWp
- ค่าไฟ 6,000 บ → ระบบ 8-10 kWp
